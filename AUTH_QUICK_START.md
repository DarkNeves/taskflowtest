## 🔄 Fluxo de Autenticação

```
Usuário acessa site
    ↓
ProtectedRoute verifica if (user)?
    ├─ Sim → Mostra Home
    └─ Não → Redireciona /login
    
Login/Sign Up
    ↓
Supabase autentica
    ├─ Erro → Mostra mensagem
    └─ Sucesso → Redireciona Home
    
onAuthStateChange
    ↓
Restaura sessão ao atualizar
    ↓
useTaskStore usa user.id
    ↓
Tarefas isoladas por usuário
```

## 🚀 Próximos Passos Opcionais

### 1. Sincronizar com Banco de Dados Supabase

Criar tabela SQL no Supabase:
```sql
CREATE TABLE tasks (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  user_id UUID NOT NULL REFERENCES auth.users(id),
  nome TEXT NOT NULL,
  desc TEXT,
  data DATE NOT NULL,
  hora TIME,
  prio TEXT,
  done BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT NOW()
);

ALTER TABLE tasks ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Users see own tasks" ON tasks
  FOR SELECT USING (auth.uid() = user_id);
```

## 🐛 Troubleshooting

### "Supabase credentials not configured"
❌ Problema: `.env.local` não preenchido
✅ Solução: Preencher `VITE_SUPABASE_URL` e `VITE_SUPABASE_ANON_KEY`

### "Auth check failed"
❌ Problema: Credenciais inválidas
✅ Solução: Copiar novamente do Supabase Dashboard

### Login não funciona
❌ Problema: Projeto Supabase não existente
✅ Solução: Criar novo projeto em supabase.com

### Tarefas sumiram após logout
✅ Esperado: Cada usuário tem suas tarefas
✅ Solução: Fazer login com a conta correta
